# Notebook

A note taking web application with a "WYSIWYG" editor, notebooks, tags, favorites, sorting and searching.

## Completed API Routes

### Users
* **GET** `/users/`
* **POST** `/rest-auth/registration`
* **POST** `/rest-auth/login/`
* **GET** `/users/{:id}`
* **PUT** `/users/{:id}/`
* **DELETE** `/users/{:id}`
